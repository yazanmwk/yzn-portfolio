## What is this?

**YZN-OS** is a playful, interactive front-end: a full-screen desktop with wallpaper, icons, a Start menu, and classic window chrome. Your work and story live inside familiar OS metaphors—folders, terminals, and dialog boxes—so visitors explore instead of scroll a long single page.

---

| | |
| --- | --- |
| **Stack** | [Next.js](https://nextjs.org/) 16 · React 19 · TypeScript · Tailwind CSS 4 |

## Feature highlights

### Boot into the experience

The first thing visitors see is not a hero section—it’s a kernel-style boot log and bold YZN OS branding. It sets the tone: this site is an OS simulation, not a generic template.

---

### Desktop, wallpaper, and welcome flow

<p align="center">
  <img src="https://yazanmwk.vercel.app/readme/desktop-welcome.png" alt="YZN-OS desktop with Welcome window" width="920">
</p>

- **Live desktop** — Wallpaper, desktop icons, and a persistent taskbar with clock and window buttons.
- **Welcome window** — Onboarding with a table-of-contents sidebar and a clear path into the desktop.
- **Start menu** — Launch apps the way you would on a classic PC.
- **Themes & display** — Tune wallpaper and appearance (including a Display Properties–style experience in Settings).

---

### Apps, games, and utilities

<p align="center">
  <img src="https://yazanmwk.vercel.app/readme/desktop-apps.png" alt="YZN-OS with Terminal and Paint open" width="920">
</p>

- **YZN Terminal** — Shell-style interface with commands, boot messaging, and room for Easter eggs.
- **MS Paint-style canvas** — Drawing tools, palette, and classic window framing.
- **Built-in programs** — Info, Projects, Contact, Settings, Algorithm visualizer, media/image tooling, and more.
- **Retro games** — DOOM, Quake, Solitaire, Minesweeper (desktop-first where noted; mobile gets sensible messaging).
- **Window manager** — Open, focus, minimize, and stack windows like a real desktop.
